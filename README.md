# fmt

Type-safe formatting libraries for Haskell.

- **stringfmt** — Core formatting as an indexed continuation profunctor, with pretty-printing via Church-encoded fixed points and recursion schemes.
- **logfmt** — `LogStr`-specialized formatting bridge for fast-logger.
- **vtyfmt** — ANSI terminal formatting via vty color types.
